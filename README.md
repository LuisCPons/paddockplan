# PaddockPlan

This is a premium, editorial landing page for a Grand Prix travel planner, bootstrapped with Next.js App Router (v16).

## Built With
- Next.js 16 (App Router)
- React 19
- Tailwind CSS v4
- Motion for React (Framer Motion)
- Lucide React

## Getting Started

First, install dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
